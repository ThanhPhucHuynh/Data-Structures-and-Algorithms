# Remove duplicates from array

## Description sorted

Given a sorted array, the task is to remove the duplicate elements from the array.

## Example

```bash
Input  : arr[] = {2, 2, 2, 2, 2}
Output : arr[] = {2}
         new size = 1

Input  : arr[] = {1, 2, 2, 3, 4, 4, 4, 5, 5}
Output : arr[] = {1, 2, 3, 4, 5}
         new size = 5

```
